--- title: Transcodificador de imagen ---

Usando el servicio de transcodificación de imágenes de App Cloud, que puede mejorar en gran medida el rendimiento de su aplicación al optimizar las imágenes para la entrega móvil.

App Cloud proporciona un servicio que puede utilizar para transcodificar imágenes para sus aplicaciones. Con el servicio de transcodificación de imágenes, puede tomar su imagen de origen y convertirla al tamaño y formato que necesita para sus aplicaciones. Por ejemplo, si tiene una imagen original de 300x300 píxeles, pero su aplicación debe mostrarla a 100x100. Puede utilizar el servicio de transcodificación de imágenes para asegurarse de que la imagen siempre tenga el tamaño correcto para su contexto, sin tener que crear un nuevo archivo de imagen para cada contexto en el que utilice la imagen. También puede recortar la imagen y especificar la calidad (relación de compresión) de las imágenes JPG. Con el servicio de transcodificación de imágenes, puede:

  • Mejore notablemente el rendimiento de su aplicación al reducir en gran medida el tamaño del archivo y el ancho de banda de red necesarios para mostrar su aplicación.
  • Reduzca la demanda de memoria al reducir el tamaño de las imágenes.
  • Asegúrese de que sus imágenes tengan el tamaño adecuado para mostrarse en dispositivos móviles.
  • Aproveche la caché del servidor de App Cloud, mejorando el rendimiento de la aplicación y reduciendo la carga en sus propios servidores.

El servicio de transcodificación de imágenes de App Cloud actúa como un proxy entre sus imágenes y sus aplicaciones. Se accede al servicio con una URL simple en la que se antepone la URL de su imagen con la URL de nuestro servicio de transcodificación. Esto le permite colocar nuestro servicio de transcodificación en sus aplicaciones con muy poco esfuerzo. Por ejemplo, si tiene una imagen que vive en http://my.example.com/image.png normalmente lo solicitaría como:

  <img  src = "http://my.example.com/image.png"  alt = "Imagen hambrienta de ancho de banda" width = "500" height = "500" />

Para transcodificarlo a una imagen más pequeña de 100x100 px, simplemente envíe la siguiente solicitud:

  <img  src = "http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&width=100&height=100"  alt = "Mi imagen esbelta" width = "100" height = "100" />

Para reducir una imagen a 100 px de ancho, conservando la relación de aspecto original:

  <img  src = "http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&width=100" width = "100" />

Para reducir una imagen a 100 px de ancho o 100 px de alto, lo que sea mayor, conservando la relación de aspecto original:

  <img  src = "http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&max_dimension=100" />

Para recortar la imagen, conservando una pieza de 100x100px en el centro, simplemente emite la siguiente solicitud:

  <img  src = "http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&crop=200,200,100,100"  alt = "Mi imagen esbelta" width = "100" height = "100" />

Para utilizar el servicio de transcodificación de imágenes, su imagen debe ser accesible en una URL disponible públicamente. La imagen debe estar en formato PNG, JPEG o GIF.

Parámetros de consulta

La llamada al servicio de transcodificación de imágenes utiliza los siguientes parámetros de consulta:

imagen (obligatorio)

La URL de acceso público en la que se puede encontrar su imagen original. Por ejemplo, image = http: //my.example.com/image.png.

formato (opcional)

El formato de la nueva imagen, que puede ser uno de los png , jpg , o gif. Si no especifica el formato, el formato de salida no se modifica de la imagen original.

Parámetros de tamaño (opcional)

Puede utilizar uno de los parámetros de tamaño, ancho , altura , o dimensión_máxima , si desea que la nueva imagen tenga un tamaño diferente al de la imagen original. Estos parámetros especifican una dimensión de la imagen final en píxeles.

  • Si no especifica ningún parámetro de tamaño, la nueva imagen tendrá el mismo ancho y alto que la imagen original.
  • Si especifica ancho , pero no altura , la nueva imagen tendrá el ancho que especifique y tendrá la altura dictada por la relación de aspecto de la imagen original.
  • Si especifica altura , pero no ancho , la nueva imagen tendrá la altura que especifique y tendrá el ancho dictado por la relación de aspecto de la imagen original.
  • Si especifica dimensión_máxima , entonces la dimensión más grande de la nueva imagen será el tamaño que especifique, y la dimensión más pequeña tendrá el tamaño dictado por la relación de aspecto de la imagen original.

En todos los casos, la nueva imagen tendrá la misma relación de aspecto que la imagen original, a menos que especifique tanto un ancho y altura y esa relación de aspecto es diferente de la relación de aspecto original.

Parámetro de cultivo (opcional)

Usar una función de utilidad de transcodificación

Para recortar su imagen, agregue el parámetro:

cultivo = x, y, w, h

Aquí, xey son las coordenadas del punto de inicio, medidas en píxeles desde la esquina superior izquierda (0,0). w y h son píxeles horizontales y verticales para conservar, comenzando en el punto de inicio y siguiendo hacia la derecha y hacia abajo.

Si se especifican tanto el recorte como el tamaño, primero se aplica el recorte y, a continuación, se cambia el tamaño de la imagen recortada.

Parámetro de calidad (opcional)

Solo para imágenes JPG, especifique la relación de compresión para la imagen resultante agregando el parámetro:

calidad = x

x es la ración de compresión que se aplicará; el valor predeterminado es 70.

Aquí hay una función de JavaScript útil para generar una URL de transcodificación de imagen:

  función getTranscodeURL (url, opciones) { var src = "http://transcode.appcloud.brightcove.com?image=" + url; if (options) { if (options.max_dimension) { src += "&max_dimension=" + options.max; } demás { if (options.width) { src += "&width=" + options.width; } si (opciones.altura) { src += "&height=" + options.height; } } si (opciones.formato) { src += "&format=" + options.format; } si (opciones.crop) { src += "&crop=" + options.crop; } si (opciones.calidad) { src += "&quality" + options.quality; } } return src; };

Por ejemplo, para transcodificar una imagen a 200 píxeles de ancho:

  var tx_url = getTranscodeURL ("http://www.example.com/example.jpg", { width: 200 });

Para recortar una imagen a 160x90, comenzando 100px desde la izquierda y 70px desde la parte superior, y especificando una relación de compresión de 65:

var tx_url = getTranscodeURL ("http://www.example.com/example.jpg", { crop: "100,70,160,90" , quality: 65 });
